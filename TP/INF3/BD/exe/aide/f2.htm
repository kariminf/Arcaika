Le contenu du logiciel
Le logiciel est un "Tp", il est orienté vers "La gestion bidagogique d'une université"

  • Inscription des étudiants dans une année universitaire.
  • Suivre des modules, chaque module a un code et un coéfficient, chaque un obtient deux notes d'examen et une note de TP par module.

Ici ce logiciel est consisté par l'établissement de la délibération d'une année (par exemple la 3ième année génie informatique).

Le but de ce TP

Le but est d'utiliser le language SQL dans un vrai projet et voir comment il fonctionne.
on donne un example:

insert into ETUDIANT
(num_e , nom_e , pre_e , age_e , adr_e)
values (un numero,'un nom','un prenom',age,'une adresse')

Ce programme fait l'insertion dant la table ETUDIANT des informations concernées par un étudiant: numéro , nom, prénom, l'age et l'adresse.

Le langage utilisé

Le language utilisé est: "Microsoft visual basic".
Comme La majorité des langages le visual basic utilise aussi l'SQL; et la cause de choisir ce langage est ça facilité de manipulation.Voici une procedure qui utilise la reqête précédente.

Sub ins_etudiant(ByVal num As Integer, ByVal nom As String, ByVal pre As String, ByVal age As Integer, ByVal adr As String)

Dim query As String
query = "insert into ETUDIANT "
query = query + "(num_e , nom_e "
query = query + ", pre_e , age_e , adr_e)"
query = query + " values (" & num & ",'"
query = query & nom & "','" & pre & "',"
query = query & age & ",'" & adr & "')"
db.Execute query
End Sub
